Look no further for a name for your next Science/Nature paper. These science paper titles are randomly generated from a Markov chain from some of the most-cited science papers.

To do this, I extracted the 12,000 articles most-cited articles from the journals Science and Nature (using expression SO=(SCIENCE OR NATURE) in Web of Science). I also created a word cloud from these titles (which shows it pays to be a cell biologist who studies expression of human protein receptors).


If you'd like to run the code yourself:

git clone https://github.com/schollz/topscience.git
python3 -m pip install bibtexparser markovify Pillow numpy
python3 run.py
